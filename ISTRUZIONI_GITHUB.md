## Fase 3 - Attiva GitHub Pages

1. Vai sul tuo repository su GitHub
2. Click su **Settings**
3. Scorri fino a **Pages** (nella sidebar sinistra)
4. Sotto **Source** seleziona: **Deploy from a branch**
5. Scegli **main** come branch
6. Folder: **/ (root)**
7. Click **Save**
